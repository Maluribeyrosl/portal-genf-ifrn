# portal-genf-ifrn
Este repositório reúne o código-fonte e os materiais do portal-genf-ifrn, um site desenvolvido para organizar, divulgar e facilitar o acesso às ações do Grêmio Estudantil do IFRN. A proposta é deixar tudo mais transparente e acessível para os alunos, mostrando de forma simples como o Grêmio funciona, quem faz parte dele e quais projetos e iniciativas estão acontecendo no campus.

1. O Grêmio Estudantil é a organização que representa os alunos dentro do campus. No IFRN Nova Cruz, ele participa das decisões escolares, ajuda a organizar eventos, promove projetos para os estudantes e faz a ponte entre a direção e a comunidade estudantil. É basicamente a voz dos alunos dentro do campus.
